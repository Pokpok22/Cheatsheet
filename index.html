Cheatsheet for CSS Syntax and Selectors

Syntax/Selector Description/Explanation
<link> Connect CSS file with HTML:
<link href="./path/to/stylesheet/style.css" rel="stylesheet" type="text/css">
Options for writing CSS 1.) Seperate file (use <link> in <head>, common practice)
2.) Write CSS in HTML file (use <style> in <head>)
3.) Inline styles (use 'style' attribute inside an element)
p The type selector selects all <p> elements
.column Selects all elements with class:'column'.
#first-item Selects element with id='first-item'.
h1, h2 Seperation with a comma applies the same CSS rule to multiple elements.
h3.section Chaining selectors (selects <h3> elements with the section class)
<div class='a b c'></div> HTML attributes can have multiple values. Values are seperated with a space. The classes of the <div> element are a, b, and c.
Specificity ID is the most specific selector, followed by class, followed by type. If property values are in conflict, the selector with the highest specifity wins.
section ol li The descendant selector targets all <li> elements that are nested in <ol> elements that are nested in <section> elements.
